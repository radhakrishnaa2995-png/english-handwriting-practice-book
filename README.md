# English Handwriting Practice Book for Kids

This project automatically generates a printable English handwriting practice book PDF.

## Book Includes

- Cover page
- How to use page
- A to Z letter practice
- Uppercase and lowercase writing
- Word practice
- Sentence practice
- 8.5 x 11 inch printable pages

## How to Generate PDF

1. Go to the Actions tab.
2. Click "Build Handwriting Practice Book".
3. Click "Run workflow".
4. Wait for the workflow to complete.
5. Download the PDF from Artifacts.
